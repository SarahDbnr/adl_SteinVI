Introduction
------------
This documentation covers the implementation of Bayesian Neural Networks (BNN) using Stein Variational Inference (SVI) with JAX. The project focuses on modularizing the code for reuse with other algorithms, like SVGD from BlackJAX.

The code is organized into modules covering various components of BNN and Stein VI implementation, including algorithms, metrics, and parameter search functionality.
